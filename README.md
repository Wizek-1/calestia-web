# Calestia Development Web

landing page website for a development agency. Built with a focus on performance, clean code architecture, and smooth user interactions.

## Tech Stack

* React 19
* Vite
* Tailwind CSS (v4)
* Framer Motion (for animations)
* EmailJS (for contact form handling)

## Project Structure

The project follows a modular component-based architecture:
* `/src/components` - Reusable UI sections (Hero, About, Process, Portfolio, etc.)
* `/src/pages` - Standalone pages (Privacy Policy, Terms of Service)
* `/src/utils` - Helper functions and configuration scripts

## Getting Started

To run this project locally

Requirements - Node.js ( https://nodejs.org/en )

1. Clone the repository

2. Install dependencies:
      ```
   npm install
      ```
3. Start local server
  
   ```
   npm run dev
   ```


## Build 

To create a ready build:


```
npm run build
```
The output will be generated in the `dist` folder.
